# Simple Music 轻松上手指南

## 🎧 这是什么？

Simple Music 是一个“打开就能用”的在线 + 桌面双端播放器：  
- **浏览器**里直接听，手机、电脑都支持。  
- **桌面版**提供 Windows / macOS / Linux 安装包，还能下载便携版拷贝带走。

## 🌍 多平台一次搜索

一个搜索框同时接入网易云、QQ 音乐、酷狗、酷我等平台，想听什么歌就搜什么，结果立即加入播放列表或收藏夹。

## 🎶 流畅播放 + 歌词同步

- 在线即播，切歌秒响应。  
- 歌词随音乐滚动，高亮当前一句，跟唱更轻松。  
- 支持循环、随机、单曲循环等播放模式，随时切换。

## ❤️ 收藏与设置自动保存

收藏夹、播放历史、音量、播放模式等都会自动记住。再次打开即可延续上次的播放状态。

## 📲 移动端也舒适

网页在手机上会自动切换成触控友好的界面，顶部集成探索/搜索按钮，底部有滑动面板，单手操作也顺畅。

## 🌈 个性化 + 探索雷达

- 深浅色主题一键切换，背景会跟随当前封面动态变色。  
- “探索雷达”会随机推荐不同风格 / 平台的歌曲，每天都有新灵感。

## 💾 桌面版额外福利

- 支持本地下载、原生文件对话框、系统托盘驻留。  
- 自动输出安装包（Setup）与绿色版压缩包（ZIP/7Z），还有 AppImage、DEB、RPM、Pacman 等 Linux 格式。  
- 更新文件（latest-*.yml、builder-debug.yml）同步发布，方便自动更新或排查问题。

## 🚀 立即体验

1. **网页端**：打开 http://nas.xdw520yq.top:2999 即可播放。  
2. **桌面端**：访问 Release 页面，选择自己系统对应的安装包或解压包。  
   - Windows：`Simple Music Setup x.y.z-arch.exe` / `simple-music-x.y.z-win_arch(-green).7z`  
   - macOS：`Simple Music-x.y.z-arch.dmg` / `simple-music-x.y.z-mac_arch.zip`  
   - Linux：`AppImage`、`DEB`（含 x64 / arm64 / armv7l）、`RPM`、`Pacman`、`tar.gz`

## ❓ 常见问题

| 问题 | 答案 |
| --- | --- |
| 需要付费吗？ | 完全免费，遵循 CC BY-NC-SA 4.0 协议。 |
| 会丢播放记录或收藏吗？ | 不会，全部自动保存。 |
| 桌面版怎么更新？ | 关注 Release 页面，下载新的 Setup 或绿色版覆盖即可。 |
| 我不会折腾电脑，能用吗？ | 可以，网页端直接打开，桌面端双击安装。 |
